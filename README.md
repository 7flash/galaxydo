## Setup

1) Start dev environment

```
git clone git@github.com:7flash/galaxydo.git
pnpm i 
npm run start
```

2) Setup configuration

[config.ts](/convex/config.ts)

3) Create voting rooms

```
curl localhost:3000/api/setupRooms
```
